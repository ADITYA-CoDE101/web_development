Frequently Asked Questions

How often is the vulnerability database updated?
ROOTSENTINEL updates its database with new CVEs as they emerge. Critical vulnerabilities are added immediately with detailed patch information.
What applications can be scanned?
We support all modern web applications including React, Vue, Angular, Node.js, Python, PHP, Java, ASP.NET, Ruby on Rails, and more.
Is my data secure?
Yes. We use enterprise-grade encryption, comply with GDPR and CCPA, and maintain SOC 2 Type II compliance. Your data is never shared with third parties.
Do you offer API access?
Yes. We provide comprehensive REST and GraphQL APIs for seamless integration with your CI/CD pipeline and development workflows.
What's included in the patch information?
Each CVE includes detailed patch recommendations, precautions, mitigation strategies, affected technologies, and links to official security bulletins.
How accurate are the scans?
Our advanced engine achieves 98.7% accuracy with minimal false positives. Continuous machine learning improves detection rates.
